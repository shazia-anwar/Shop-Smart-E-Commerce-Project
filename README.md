# Shop Smart E-Commerce Project

## Project Overview
This project focuses on building a machine learning classification model using e-commerce customer data.

A structured preprocessing pipeline was implemented to handle numerical and categorical features, followed by training a Decision Tree classifier.

---

## Technologies and Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- JupyterLab

---

## Key Concepts Implemented
- Train-Test Split
- ColumnTransformer for handling different feature types
- StandardScaler for numerical features
- OneHotEncoder for categorical features
- Pipeline for structured workflow
- DecisionTreeClassifier for classification
- Model evaluation using:
  - F1 Score
  - Confusion Matrix
  - Classification Report

---

## Machine Learning Workflow
1. Data Preprocessing  
2. Train-Test Split  
3. ColumnTransformer Implementation  
4. Pipeline Creation  
5. Model Training  
6. Model Evaluation  

---

## Model Evaluation
The model was evaluated using F1 Score, Confusion Matrix, and Classification Report to measure classification performance effectively.

F1-score was selected to balance precision and recall.

---

## Key Outcome
The Decision Tree model successfully classified customer behavior.  
Using a preprocessing pipeline ensured consistent transformations and reduced the risk of data leakage.
